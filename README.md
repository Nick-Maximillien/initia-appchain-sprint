# Initia Appchain Sprint

This repository contains the foundational code for our submission to **INITIATE: The Initia Hackathon**.

## Hackathon Details
- **Focus**: Web3 Appchains and blockchain infrastructure.
- **Goal**: Launch apps with real revenue and ownership.
- **Submission Page**: [INITIATE on DoraHacks](https://dorahacks.io/hackathon/initiate/detail)
- **Deadline**: April 15, 2026

## Project Structure
- `app.py`: The foundational Flask service.

## Getting Started
```bash
pip install flask
python app.py
```
